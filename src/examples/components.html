Image container

The image-fill class will make sure the image fills its bounds by centering, scaling and cropping it, no matter the aspect ratio.
The image-border class, if added to a component which contains the image, adds a nice 'photo' border and shadow.

Zoom effects

Add the zoom class to any component to add a zoom effect when hovering.

Use a container with zoom-inside to add an internal zoom effect when hovering.
