Grid system

The responsive/fluid grid system (build on flexbox) allows to segment the UI into areas which span 1 to 12 view columns.
This ensures your content is properly aligned, regardless of the screen size. The grid spans 12 columns on large screens, and is reduced stepwise for smaller screen sizes (responsive) to 6,3,2 and 1 column.
