Grid system

The responsive/fluid grid system (build on flexbox) allows to segment the UI into areas which span 1 to 12 view columns.
  • Fluid:
    Columns resize fluidly, ensuring your content is properly aligned, regardless of the grid size.
  • Responsive:
    The grid spans 12 columns on large screens, and is reduced stepwise for smaller screen sizes (responsive) to 6,3,2 and 1 column.
  • Gutters:
    The grid columns can have optional gutter (space between columns) which is uniformly distributed.
