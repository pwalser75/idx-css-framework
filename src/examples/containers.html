Table layout

Use the table-layout class on a container, to format it as a table. Its children will be rows, and their children be the cells of the table.

Lists

Use the list class to format any ul or div as a list. Use the list list-icons classes to format the list with leading icons. List items can have icons, annotated with the class list-icon.
